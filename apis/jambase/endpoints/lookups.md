# Lookups

Reference endpoints that return available data sources for each entity type. No parameters required.

## GET `/lookups/event-data-sources`

Returns all supported event data sources:

`axs`, `dice`, `etix`, `eventbrite`, `eventim-de`, `jambase`, `seated`, `see-tickets`, `see-tickets-uk`, `sofar-sounds`, `seatgeek`, `suitehop`, `ticketmaster`, `tixr`, `viagogo`

## GET `/lookups/stream-data-sources`

Returns all supported stream data sources (e.g., `nugs.tv`, `jambase`).

## GET `/lookups/artist-data-sources`

Returns all supported artist data sources:

`axs`, `dice`, `etix`, `eventbrite`, `eventim-de`, `jambase`, `seated`, `seatgeek`, `spotify`, `ticketmaster`, `viagogo`, `musicbrainz`

## GET `/lookups/venue-data-sources`

Returns all supported venue data sources:

`axs`, `dice`, `etix`, `eventbrite`, `eventim-de`, `jambase`, `seated`, `seatgeek`, `suitehop`, `ticketmaster`, `viagogo`
